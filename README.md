# Chatsapp

Chat application using NodeJS and React

## Client

Using the following tools:
* query-string
* react
* react-dom
* react-emoji
* react-router
* react-router-dom
* react-scripts
* react-scroll-to-bottom
* socket.io-client

Deployed on Netlify. 
Live link: https://chatsapp-client-rhuan.netlify.app/

## Server

Using the following tools:
* cors
* express
* nodemon
* socket.io

Deployed on Heroku:
Live link: https://shielded-cove-98428.herokuapp.com/
